📨 Telegram Integration

Connect your Telegram bot to automatically receive generated videos. Bot also works 24/7 — send prompts directly to your bot even when this website is closed!

How to set up:
1. Open Telegram, search for @BotFather, send /newbot and follow steps to get your Bot Token
2. Start your new bot (send it /start)
3. To get your Chat ID: send any message to @userinfobot — it will reply with your ID
4. Paste both values above and click Save

Bot Commands:
/video <prompt> — Generate video
/batch — Start batch mode (send prompts, then /go)
/image <prompt> — Generate image
/enhance <prompt> — Enhance prompt (2x)
/random — Generate random prompt
Or just send any text as a video prompt!
